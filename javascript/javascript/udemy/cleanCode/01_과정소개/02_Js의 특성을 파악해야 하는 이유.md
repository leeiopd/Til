# Udemy - CleanCode

### 1. 과정 소개

##### 02_Js의 특성을 파악해야 하는 이유

* 자바스크립트는 역사적으로 굉장히 많은 변화를 이뤄온 언어
  * 언어를 모두 이해하기 쉽지 않음
  * 특성과 맥락을 파악하면, 향후 단번에 이해할 수 있는 백그라운드를 가질 수 있다.

* 자바스크립트 특성
  * 몽키패치
    * 몽키패치 : 런타임 중신 프로그램의 내용이 변경되는 행동을 의미
    * 안티패턴 이라고도 함
      * 소스코드가 의도대로 동작하지 않음

  * 문법
    * Java 와 유사

  * 문자열 / 배열 / 정규표현식
    * 펄과 유사

  * 함수
    * 오크와 유사

  * 클로저 / 스코프 환경
    * 스키마와 유사

  * 프로토타입
    * 셀프와 유사

  * 이벤트
    * 하이퍼토큰과 유사
